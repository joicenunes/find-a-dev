# Projeto criado com [Create React App](https://github.com/facebook/create-react-app)

## No que consiste a aplicação

Nela você pode pesquisar nomes de usuários do github. Caso o usuário exista, são exibidas informações básicas de acordo com a disponibilidade, como nome, localização, biografia. Você também encontra um link direto para o perfil do usuário no próprio github e uma lista dos repositórios favoritos do usuário solicitado.

## Como testar

Após clonar o repositório na sua máquina local e instalar as dependências, rode o script (necessita node.js com npm) para iniciar a aplicação:

```
npm run start
```

O script vai rodar a aplicação na porta 3000.
